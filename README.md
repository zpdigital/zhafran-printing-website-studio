# ZHAFRAN Printing & Website Studio

Website statis siap upload ke GitHub Pages / Vercel / Netlify.

## Struktur
- `index.html` halaman utama
- `pages/undangan-digital.html` daftar tema undangan digital
- `pages/reseller.html` halaman reseller
- `css/style.css` styling utama
- `js/data.js` data produk dan tema
- `js/main.js` fungsi render, filter, menu, animasi
- `assets/logo.jpg` logo

## Cara upload GitHub Pages
1. Buat repository baru di GitHub.
2. Upload semua file dan folder di project ini.
3. Masuk Settings > Pages.
4. Source pilih `Deploy from a branch`.
5. Branch pilih `main`, folder `/root`.
6. Save.
